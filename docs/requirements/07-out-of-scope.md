# 7. 明确不做的范围（Out of Scope）

以下功能在多轮需求讨论中被明确排除，**MVP 与所有规划阶段均不实现**。如未来需要实现，需重新进入需求分析流程。

## 7.1 Agent 能力相关

- **角色化预定义子 agent**：不内置 code-explorer / code-reviewer / test-writer 等固定角色 agent；不支持 `~/.mini-agent/agents/*.md` 之类的角色定义机制。
- **子 agent 嵌套派生**：子 agent 不允许再派生子 agent。
- **异步子 agent / Team 模式**：所有子 agent 调用均为同步阻塞，不支持长时间后台运行的 team 协作。
- **跨会话长期记忆**：不实现全局 memory.md 之类的跨会话记忆机制。

## 7.2 会话与数据管理

- **会话导出**：不支持将会话导出为 markdown 或其他格式的功能。
- **会话备份 / 跨设备同步**：不实现。
- **会话搜索**：不实现全文检索（用户通过 `/sessions` 列表浏览即可）。

## 7.3 Web UI 与服务化

- **Web UI 鉴权**：不引入登录、Token、OAuth 等鉴权机制。Web UI 默认仅供本地用户使用。
- **多用户 / 多租户**：本产品按单用户本地工具定位，不考虑多用户隔离。
- **远程部署形态**：不提供 SaaS、容器化、企业部署相关需求。
- **MCP Server**：仅做 MCP Client，不做 MCP Server。

## 7.4 用户体验与扩展

- **CLI 富渲染**：CLI 不做 markdown / 代码高亮 / diff 等复杂渲染；这些渲染只在 Web UI 提供。
- **用户级斜杠命令扩展**：用户不可自定义新的斜杠命令。
- **运行时代码插件机制**：除 MCP Client 之外，不支持运行时加载用户代码插件 / WASM / 外挂二进制。Skill 模块（详见 `11-skill.md`）**不视为代码插件**：skill 是模型可读取的领域知识包 + 受 `bash` 权限管控的脚本，不能定义新工具、新子 agent 或新斜杠命令。
- **UI 主题与多语言**：暂不考虑（不属于本期讨论范围）。

## 7.5 Agent Loop 内部机制

- **`finish` 工具**：不引入显式终止工具，循环依赖自然结束 / 步数上限 / 用户中断三种条件。

## 7.6 其他未提及的常见能力

以下是常见但未在本次讨论中确认需要的能力，统一视为不在范围内；如需启用须经新一轮需求评审：

- 多用户协作 / 共享会话
- 任务调度 / 定时执行
- 通知（邮件、Webhook）
- 遥测 / 用量上报
- 国际化 / 无障碍
- 移动端支持
- 离线模式
- 数据备份与还原
- 版本迁移工具
- 插件市场
